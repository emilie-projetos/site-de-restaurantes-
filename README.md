# Site de Restaurantes

Projeto desenvolvido com HTML e CSS com o objetivo de praticar desenvolvimento web.

##  Tecnologias utilizadas
- HTML
- CSS

## Objetivo
Criar uma interface de site para restaurante, com seções organizadas e apresentação de cardápio.

##  Funcionalidades
- Página estática
- Estrutura de menu
- Organização visual de conteúdo

##  Como executar
Abra o arquivo index.html no navegador
